This is a simple practice project built using Express.js and Axios to learn how to fetch and display data from an external API.

The application fetches random activities from the Bored API and displays useful information such as activity name, type, and number of participants.

This project was created purely for practice purposes to understand backend concepts and API integration.

## Features

- Fetch random activity from API
- Filter activities based on type and participants
- Display activity details dynamically using EJS
- Error handling when no activity is found
- Server-side rendering using Express.js

## Tech Stack

- Node.js
- Express.js
- Axios
- EJS
- HTML
- CSS

## API Used

Bored API:
https://bored-api.appbrewery.com/
